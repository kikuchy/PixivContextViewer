    pixiv context viewer

    〜これまでにない pixiv 体験を、あなたに〜

    pixiv context viewer (pcv)は、これまでに無い操作性をあなたに提供するブラウザプラグインです。 pcvを入れることによって、あなたの pixiv 体験は大きく変化するでしょう。

    • 特徴
      • イラスト詳細画面に、以下の一覧を表示します。
        • 各種ランキング一覧（デイリー、オリジナル、etc）
        • 新着一覧
        • お気に入りユーザーの新着一覧
        • マイピクの新着一覧
        • 検索結果
      • 一覧は、ユーザーが閲覧しているページにあわせ、自動的に内容が変わります。例えば、デイリーランキングを見ている時は、一覧にデイリーランキングを表示します。
      • 多数のブラウザで使用可能です。
      • オープンソースなので、無料で使用できます。
    • 用件
      • 最新版のGoogle Chrome
      • 最新版のOpera（対応予定）
      • 最新版のSafari（対応予定）

    pcv for Google Chrome

    インストール

    1. pcv for Google Chrome をダウンロードします。
    2. Google Chrome を起動して、アドレスバーにchrome://extensions/と入力します。
    3. Google Chrome の画面の中に、ダウンロードした crx ファイルをドラッグ＆ドロップします。
    4. 「追加」ボタンを押します。

    おめでとうございます！　あなたは今、新しい pixiv 体験のための一歩を踏み出しました！ 早速 pixiv にアクセスして、ランキングを見てみてください。

    特徴

    これまでに無い閲覧体験

    pcv を使えば、面倒で鬱陶しい画面遷移がなくなります！
    しかも pcv をスクロールするだけで、次々にイラストが読み込まれます。

    pcv は、これまでの pixiv では面倒だった、あなたのブラウジングを快適にします。 おそらくあなたは、これまで以下のようにしてお気に入りの絵を見つけていたでしょう。

    1. ログイン
    2. 「お気に入りユーザーの新着作品」をクリック
    3. 最新のイラストをクリック
    4. 戻る
    5. 次の絵をクリック
    6. 戻る ......

    これまでの常識は捨ててください。今が、ブラウザの「戻る」ボタンと別れの挨拶をするときです。
